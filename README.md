### I've fixed up and fully revived the 0.5 beta version of Call of Mini Dino Hunter with some improvements to make it a lot more enjoyable without changing the feel of the game, below are all the changes:

##

- **Bug fixes**

- **Map fixes**

- **Visual fixes** 

- **Added all the missing audio**

- **Improved visuals**
[the textures and everything are still the same, they just have
better compression and the mipmaps look significantly better.]

- **Improved performance**

- **Everything is fully obainable now, as originally skills and
characters were impossible to obtain.**

- **Ported the game to unity 6**

- **PC Support for Windows and Linux**

- **Android 16/64 bit support**

## Here are all of the PC Controls:

- **Movement: WASD**
- **Attack: Left Click**
- **Toggle Cursor/Mouse Lock: Shift + F1**
- **Change Weapons: Q for previous weapon, E for next weapon**
- **Skill: 1**
- **Increase Sensitivity: O**
- **Decrease Sensitivity: I**
