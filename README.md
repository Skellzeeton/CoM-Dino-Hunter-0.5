## Here are all of the PC Controls:

- **Movement: WASD**
- **Attack: Left Click**
- **Toggle Cursor/Mouse Lock: Shift + F1**
- **Change Weapons: Q for previous weapon, E for next weapon**
- **Skill: 1**
- **Increase Sensitivity: O**
- **Decrease Sensitivity: I**
